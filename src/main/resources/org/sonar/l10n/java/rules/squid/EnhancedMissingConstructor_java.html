Checks that classes (except abstract ones) define a constructor and don't rely on the default one, unless the class has one or more 'mitigating' annotations.

Noncompliant Code Example

class MyClass {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

Compliant Solution

class MyClass {
    private MyClass() {}
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
or 
@SpringBootApplication
class MyClass {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
